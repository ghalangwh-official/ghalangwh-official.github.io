## 🎮 Features

### 🌟 Visual Effects
- **Matrix Rain Background** — Animated green character rain (Canvas-based)
- **ASCII Art Logo** — Giant "GHALANG" with glow effect
- **System Info Bar** — Live clock, online status, location, system info
- **Terminal Window** — macOS-style buttons, custom scrollbar, blinking cursor
- **Cyberpunk Theme** — Matrix green (#00ff41) on deep blue-black

### ⌨️ Interactive Terminal

**12+ Commands:**
```bash
help        # Show all commands
about       # About me & mission
projects    # List featured projects with status
skills      # Full tech stack breakdown
contact     # Email & links
social      # Social media
stats       # GitHub achievements timeline
repos       # Quick repo access guide
whoami      # System information
quote       # Random hacker quotes
matrix      # Toggle matrix effect
open <repo> # Direct link to repo (jadwal, crawler, wallpaper, profile)
clear       # Clear terminal
```

**Features:**
- ✅ Tab completion (type partial command, press Tab)
- ✅ Command history in output
- ✅ Smooth scrolling
- ✅ Mobile responsive

## 🎨 Design Highlights

### Color Scheme
- **Background:** `#0a0e27` (deep blue-black)
- **Primary:** `#00ff41` (Matrix green)
- **Text:** `#e5e7eb` (light gray)
- **Accents:** `#64748b` (slate)

### Animations
- `fadeIn` (1s) — Header fade-in
- `slideDown` (0.8s) — System info slide
- `slideUp` (1s) — Terminal entrance
- `glow` (2s loop) — Text glow effect
- `pulse` (2s loop) — Status badge
- `blink` (1s loop) — Cursor blink

### Typography
- **Font:** Courier New (monospace)
- **ASCII Art:** Optimized for readability
