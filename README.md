# Ghalang's Modern Portfolio

Modern GitHub Pages portfolio with a premium dark UI, selected projects, and a lightweight interactive console.

## What it shows

- GitHub profile and contact links
- Selected public repos
- Core stack and working principles
- A small terminal console for quick navigation

## Design direction

- Dark premium interface
- Strong typography and spacing
- Subtle motion, no noisy gimmicks
- Mobile-friendly layout
- Vanilla JS and CSS only

## Commands

```bash
help
about
projects
stack
contact
signal
quote
clear
open <repo>
```

- Jadwal Sholat

- ACA Civilization OS

## Tech

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages

## Notes

This site is intentionally static and low maintenance.
The interactive console is there for navigation and personality, not as a dependency-heavy app shell.
